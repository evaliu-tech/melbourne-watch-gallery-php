# Melbourne Watch Gallery PHP

A PHP and MySQL e-commerce website for browsing smart watches and managing product information.

## Project Description

Melbourne Watch Gallery is a responsive e-commerce website developed with PHP and MySQL. Users can browse products, view product details, and add products to a shopping cart. Admin users can log in to manage product records, including adding, editing, and deleting products.

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- PHP
- MySQL
- XAMPP
- phpMyAdmin

## Features

- Product listing page
- Product detail page
- Shopping cart using localStorage
- Admin login system
- Product management page
- Add product
- Edit product
- Delete product with confirmation
- MySQL database integration
